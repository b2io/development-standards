# HTML Style and Conventions

### HTML Structure:

- Use semantic HTML elements like `<header>`, `<nav>`, `<main>`, `<section>`, `<article>`, and `<footer>` to enhance accessibility and SEO.

- Use consistent naming conventions for HTML elements and attributes.

- Employ single-word syntax for boolean attribute values to enhance readability, minimize visual clutter.

### Accessibility:

- Ensure all images have descriptive alt attributes for screen readers.

- Use appropriate ARIA roles and attributes to enhance accessibility.

### Attributes:

- Minimize the use of inline styles and prioritize external CSS for styling.
